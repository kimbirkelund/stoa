# Projects

Feature code: **`PJ`** (requirement IDs `RPJ-*`; see [conventions](../../conventions.md)).

The **Projects** feature owns how the [projects](../../definitions.md) of an open
workspace are presented and navigated — the active projects appear as
browser-like tabs, and a project is active exactly when its tab is open.

- [Description](description.md) — how project navigation works (prose, mental model).
- [Requirements](requirements.md) — behavioral requirements (`RPJ-*`) and open seeds.

[cross-cutting requirements](../../cross-cutting.md) (X-1 keyboard, X-2 mouse),
and [definitions](../../definitions.md).
